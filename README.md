# EnguageTx iOS SDK

## Getting Started

### CocoaPods
```
platform :ios, "8.0"
source 'https://gitlab.com/engaugetx/EngaugeTxPodSpecs.git'

target 'EnguageTxSampleIosApp' do
  use_frameworks!
  pod 'EngaugeTx', '~> 0.0.1'
end
```
